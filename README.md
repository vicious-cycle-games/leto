# Leto
Leto is a first-person desktop driving game. **WIP REPOSITORY**

## Support
All operating systems that we plan to support in the future are below. Those marked with (yes) are currently supported, (x) are not.

- Windows
	- x86_64 (yes)
	- i836 / x86_86 (x)
- Linux
	- x86_64 (yes)
	- i836 / x86_86 (x)
	- arm64 (x)
- MacOS (x):
	- x86_64 (x)
	- arm64 (x)
